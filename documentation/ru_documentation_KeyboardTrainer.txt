Основная логика проекта находится в классе Model.cs (KeyBoardTrainer/Models/Model.cs), методы:
NewRound(str_input) - метод для инициализации нового раунда для тренировки печати, строковой параметр str_input - это символы, которые пользователь должен ввести с клавиатуры чтобы завершить раунд.
SendChar(chr) - метод, который следует вызвать когда пользователь нажимает кнопку на клавиатуре, строковой параметр chr - это символ который дает нажатая клавиша.
Поля:
ChrsLeft - символы, которые пользователю осталось ввести для завершения раунда.
События:
Mistaked - событие, которое вызывается когда пользователь вводит не тот символ который от него ожидался.

Класс Updater.cs (KeyBoardTrainer/Models/Updater.cs) служит для обновления программы с репозитория на github.com, методы:
NeedUpdate - метод, который сравнивает текущую версию продукта и версию в репозитории на github.com.
Update - метод, который скачивает обновленную версию продукта если локальная версия и версия с репозитория различаются и запускает скаченный файл, закрывая программу.
Поля:
ThisVersion - локальная версия продукта.

Класс Loc.cs (KeyBoardTrainer/Models/Loc.cs) служит для хранения переводов строк(английский, русский), методы:
AddTranslate(eng, rus) - добавляет в хранилище переводов английскую строку и её перевод.
Translate(engOrRus) - возвращает перевод строки engOrRus(входная строка на любом из языков).
Поля:
Curr_Language - на какой из языков переводить (использует метод Translate).

Перечисление MLanguage.cs (KeyBoardTrainer/ViewModels/MLanguage.cs) хранит доступные языки программы.

Класс Statistics.cs (KeyBoardTrainer/ViewModels/Statistics.cs) служит для передачи параметров в события, поля:
CharsLeft - строковое поле, символы, которые пользователю осталось ввести для завершения раунда.
Mistakes - числовое поле, количество ошибок допущенных пользователем за этот раунд.
Speed - числовое поле, скорость набора пользователя.
Time - количество времяни которое прошло с момента начала раунда.

Основная прослойка между отображением Views и Models это ViewModel.cs (KeyBoardTrainer/ViewModels/ViewModel.cs), методы:
AddTranslate(eng, rus) - добавить перевод для английского слова (вызывает AddTranstale из Loc.cs).
Translate(engOrRus) - возвращает перевод слова (вызывает Translate из Loc.cs).
ChangeLanguageTo(language) - изменить язык.
Update() - спрашивает пользователя хочет ли он обновиться и обновляет если нужно (использует Updater.cs).
SendChar(chr) - сообщает модели что была нажата кнопка, chr - символ который получается при нажатии кнопки.
NewRound(updateTime) - запускает новый раунд со случайными символами(символы зависят от языка), updateTime - обновлять ли время начала раунда.
NewRound(input, updateTime) - перегрузка NewRound(updateTime), вместо случайной строки использовать свою.
Поля:
CharsLeft - строка которую нужно ввести пользователю для завершения раунда.
Begin - время начала раунда.
События:
StatisticChanged - вызывается при изменении статистики, в качестве параметра Statistics.cs
